# 🤖 로보틱스 & 회로 시뮬레이션 AI 튜터 (MVP)

## 프로젝트 개요
자연어 명령만으로 회로도(Ngspice)와 로봇 제어 코드(PyBullet)를 자동 생성하고, 가상 환경에서 즉각적인 실행 검증 및 시각화 피드백을 제공하는 대화형 AI 에이전트입니다. 교내 학생들의 하드웨어 및 소프트웨어 융합 교육 실증용으로 설계되었습니다.

## 아키텍처 및 워크플로우 (안티그래비티 CLI 기반)
1. **자연어 입력:** 사용자가 챗 인터페이스에 "5V 전원에 저항과 LED를 연결해 줘"라고 입력.
2. **Harness & Routing:** 안티그래비티 하네스가 인텐트를 파악하여 `circuit_bot` 서브에이전트로 작업 할당.
3. **Ghost Runtime 실행:** 에이전트가 백그라운드에서 `ngspice` MCP 툴을 호출하여 넷리스트 연산 수행.
4. **시각화 및 피드백:** Matplotlib 기반의 정적 그래프를 생성한 후, 튜터 에이전트가 학생 눈높이에 맞춘 해석을 제공.
